Questions Editor (JSON)

Edits save to `content/criteria.json` via the local dev server.

Help

Use this editor to override criteria with matching id values.

Format:

{
  "criteria": [
    {
      "id": "LWP-S1",
      "title": "LWP-S1: Wellness Policy Coordination Team",
      "criterionText": "...",
      "questions": [
        {
          "id": "Q1",
          "text": "Question text",
          "type": "boolean | single | multi | evidence",
          "options": ["Yes", "No"],
          "showIf": [{ "id": "Q1", "equals": "Yes" }]
        }
      ],
      "scoring": {
        "hardFail": [],
        "required": [],
        "eligibility": ""
      }
    }
  ]
}

Rules can also be set per question via showIf using equals or includes.
